Hacktoberfest FAQ

Q: What is Hacktoberfest?
A: Hacktoberfest is a month-long celebration of open source software. It encourages people to contribute to open source projects on platforms like GitHub and rewards them with swag and prizes.
Q: How do I participate in Hacktoberfest?
A: To participate in Hacktoberfest, you need to:
  1. Register on the Hacktoberfest website.
  2. Find open source projects on GitHub.
  3. Make at least four valid pull requests in October.
  4. Check your progress on the Hacktoberfest website.
Q: What qualifies as a valid pull request?
A: A valid pull request for Hacktoberfest should:
  • Be made in a public repository on GitHub.
  • Include meaningful changes (e.g., code improvements, bug fixes, documentation).
  • Be accepted or merged by the project maintainers.
Q: What are the rewards for participating?
A: Rewards may vary from year to year, but typically include t-shirts, stickers, and other Hacktoberfest swag. Check the Hacktoberfest website for the current year's rewards.
